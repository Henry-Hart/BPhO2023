This website is our entry for the challenge. We have attempted all the challenges and extension material, with a focus on customization and usability.

We used multiple graphing libraries and pure JavaScript to power our graphs. For more information on our choices, please view the implementation details section.

To navigate the challenges, please use the menu on the left. On each challenge, there is an info box ? which displays further information about the challenge.
